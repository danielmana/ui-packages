# UI Core - Installation

<p class="description">Install UI Core</p>

UI Core is available as an [npm package](https://www.npmjs.com/package/@danielmana/ui-core).

## npm

To install and save in your `package.json` dependencies, run the command below using **npm**:

```sh
npm install @danielmana/ui-core
```

<!-- #react-peer-version -->

Please note that [react](https://www.npmjs.com/package/react) >= 17.0.0 and [react-dom](https://www.npmjs.com/package/react-dom) >= 17.0.0 are peer dependencies.

## SVG icons

In order to use prebuilt SVG Material icons, such as those found in the [icons demos](/ui-core/icons/)
you must first install the [@danielmana/ui-icons](https://www.npmjs.com/package/@danielmana/ui-icons) package:

<!-- #default-branch-switch -->

```sh
npm install @danielmana/ui-icons
```
